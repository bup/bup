# NAME

bup-validate-ref-links - check repository refs for links to missing objects

# SYNOPSIS

bup validate-ref-links [*ref*...]

# DESCRIPTION

This command is equivalent to `bup validate-refs --links`. Please
prefer that.

`bup-validate-refs`(1)
